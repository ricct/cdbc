About CDBC

CDBC can provide proxy access to Cloud Database. It consists of CDBC Client and CDBC Service.CDBC Client connect to CDBC Service which can provide proxy access to Database. CDBC Service use WCF and run on Microsoft IIS so Windows instance on cloud is needed.
