About connecting to Oracle

CDBC client offers easy access to remote Oracle database from .NET applications. It has the similar ADO.NET architecture as ODP.NET. Code with familiar classes , properties and methods. But it does not support all functions of ODP.NET. Supported properties and methods are listed below.
